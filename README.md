# zsh-autozcalc
A Zsh plugin that detects math expressions and evaluates them

## Installation
If you don't use any plugin manager like [Zinit](https://github.com/zdharma-continuum/zinit), simply source the `zsh-autozcalc.plugin.zsh` file in your `.zshrc` configuration file.

**But if you do**, add `upslaught/zsh-autozcalc` prefixed by your plugin manager's command to load plugins, for example:
```sh
zinit load upslaught/zsh-autozcalc
```
